Data Folder Overview
This folder contains the datasets and certificate documents used for blockchain-based academic certificate verification.
Contents

EXCEL SHEET PROJECT 09 GROUP 41.xlsx  
Dataset of student details, degrees, universities, certificate IDs, issue dates, and CGPA values. This file is the primary input for data cleaning, hashing, and blockchain verification.

Certificate example.pdf  
Sample academic certificates issued by different universities. These are used to demonstrate how blockchain verification detects authenticity and tampering.

Usage
Use the Excel dataset for data cleaning and hash generation.
Compare certificate details in the Excel file with the PDF samples to simulate real-world verification.
The cleaned dataset and certificate hashes are later stored on the blockchain for authenticity checks.
