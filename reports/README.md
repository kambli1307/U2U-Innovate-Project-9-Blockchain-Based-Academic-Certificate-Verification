Demo Video (Demo.mp4)
A walkthrough of the code execution and workflow:
User uploads a certificate.
Server processes request and generates hash.
Hash is stored and verified against blockchain.
AI model detects authenticity and fraud.
Verification result is displayed to the user.

Objective
To provide a secure, transparent, and tamper-proof system for verifying academic certificates using Blockchain technology and AI-based fraud detection.
